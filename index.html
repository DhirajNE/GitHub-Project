HTML/GitHub Project

By means of this HTML and GitHub project, I acquired useful knowledge in contemporary web development techniques and version control systems. Building a responsive website from scratch improved my knowledge of semantic HTML5 elements and CSS Grid layouts, thereby matching with Ryou and Ryu (2018) ideas of usability. Working with GitHub helped me to better understand collaborative development techniques, especially the relevance of commit messages and branching strategies as stressed by Constantino et al. (2020).

Key learnings:

  • Implemented responsive design principles using CSS Grid and Flexbox
  • Mastered basic Git workflows including commits, branches, and merges
  • Applied semantic HTML5 elements for better accessibility and SEO
  • Learned the importance of mobile-first design approaches

Typography Project

This project increased my respect of typeography as a fundamental component of web design. Inspired by Bringhurst's (2004) ideas of typographic brilliance, I tried several typefaces, spacing, and hierarchies to produce readable and visually appealing layouts. The project clarified for me how typeface shapes brand impression and user experience.

Key discoveries:

  • Explored the relationship between typeface choices and content tone
  • Learned proper spacing techniques for optimal readability
  • Practiced creating effective typographic hierarchies
  • Understanding the importance of responsive typography

Reflective Essay

Over the Design Fundamentals course this semester, I have had a transforming experience learning and using web and graphic design concepts. Over the three primary projects—HTML/GitHub, CARP, and Typography—this reflection article investigates my development and learning experience.

The HTML/GitHub project started my path and exposed me to the technical underpinnings of web development. As Ryou and Ryu (2018) highlight, modern web development depends on a knowledge of semantic HTML5 elements. Though first I found version control ideas difficult, as Constantino et al. (2020) emphasise, actual application helped me to value how Git processes enable group development. This project helped me to realise that, just as much as visual design is crucial, clean, orderly code is.

The CARP project fundamentally changed the way I saw design concepts. Williams' 2015 Contrast, Alignment, Repetition, and Proximity paradigm gave me a methodical approach to visual design. I found that these ideas are useful tools rather than only abstract ideas that may turn everyday layouts into striking visual experiences. The most important lesson was realising how these components interact cooperatively; for example, how good alignment supports proximity groups and how regulated repetition could improve rather than weaken contrast.

Originally underappreciated, typeography turned out to be essential component of web design. Bringhurst's 2004 observations on typeographic principles let me see the subtle but significant influence of font selection and spacing. Experimentation helped me to discover that typeography is about generating mood, establishing hierarchy, and improving user experience rather than only readability. From overall font matching techniques to the smallest letter spacing, this project forced me to consider design at both macro and micro levels.

Developing an integrated knowledge of how technical abilities and design concepts complement each other has been among the most important results of this course. HTML structure, CARP ideas, and typeographic expertise taken together have let me approach web design holistically. Good web design is about developing useful, easily navigable, aesthetically pleasant experiences that meet user needs, not only about how things look.

Looking forward, I see places for more development. I wish to investigate more complex CSS techniques and responsive design patterns even though I have confidence in applying design ideas. Since these are so important for producing really inclusive web experiences, I also want to learn more about user experience design and accessibility criteria. Equipped with both technical abilities and design ideas, this course has set a strong basis for my career in web design. Above all, it has shown me that design is an ongoing learning, trying, and refining process that keeps on outside the classroom.
